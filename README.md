> **Developed with ❤️ and 🧠 by Team Zombies**

An advanced, fully Object-Oriented implementation of the classic *Plants vs. Zombies* game. Developed as a comprehensive project for the Advanced Programming course at Sharif University of Technology (SUT), this project emphasizes clean architecture, robust design patterns, and stable data management.

---

## 🌟 Key Features

* **MVC Architecture:** Strict separation of game logic (Model), user interface (View), and input handling (Controller) ensuring high maintainability and readability across a large-scale codebase.
* **Design Patterns:** Strategic implementation of core software design patterns:
  * `Factory` for cleanly instantiating various plant and zombie types.
  * `Strategy` for dynamic attack mechanisms and entity behaviors.
  * `Singleton` for global game state management.
  * `Builder` for constructing complex game entities step-by-step.
* **Big Wave Beach Map:** Accurate simulation and logic implementation of the Big Wave Beach environment, complete with distinct water mechanics and localized grid challenges.
* **SQLite Database Integration:** Seamless database management using SQLite for executing queries, saving player data, tracking level progression, and securely loading user profiles.
* **Scalable Codebase:** Designed to support extensive feature expansions, effectively managing thousands of lines of code while strictly adhering to SOLID principles.

---

## 🛠 Tech Stack

* **Programming Language:** Java
* **Database Management:** SQLite (JDBC)
* **Paradigm:** Object-Oriented Programming (OOP)
* **Architecture:** Model-View-Controller (MVC)

## 🚀 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites
* [Java Development Kit (JDK)](https://www.oracle.com/java/technologies/javase-downloads.html) (Version 11 or higher recommended)
* SQLite JDBC Driver configured in your project's build path/dependencies

### Installation & Execution
Clone the repository:
